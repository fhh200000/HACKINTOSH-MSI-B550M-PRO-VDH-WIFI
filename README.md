Hackintosh for MSI B550M PRO-VDH WiFi Board. 
# Hardware
* MSI B550M PRO-VDH WiFi  
* AMD Ryzen 5 5600X  
* 16GB Memory  
* Geforce GTX 1050Ti 4GB  
# Version
10.13 High Sierra
